# Practical Microcontroller Architecture

A long-term learning and writing project by Simul Ahmed.

This project is built to deeply understand microcontrollers as an embedded systems engineer, not just to collect disconnected notes. The goal is to create a clear, practical, engineering-focused book that explains how microcontrollers work, how firmware controls hardware, how different MCU families compare, and how to choose the right MCU for real systems.

## Project Goals
- Build a deep and transferable understanding of microcontrollers
- Study MCU architecture, firmware control, peripherals, and system design
- Learn major MCU families in a structured order
- Keep broad awareness of the full MCU ecosystem, from low-cost Chinese MCUs to high-performance industrial families
- Gradually turn the learning journey into a complete technical book

## Reading Structure
- `BOOK_OUTLINE.md` -> full book map
- `MCU_STUDY_PLAN.md` -> full MCU family roadmap
- `CURRENT_STATUS.md` -> where the study currently is
- `book/` -> actual book chapters
- `mcu/` -> MCU-family-specific overview pages

## Core Study Philosophy
The goal is not to memorize every MCU. The goal is to understand how any MCU works so that a new family can be learned quickly, compared intelligently, and applied in real engineering work.
