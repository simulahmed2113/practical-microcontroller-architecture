# MCU Family Notes

This folder contains family-specific MCU overview pages. These are not the main book chapters. Instead, they act as focused ecosystem notes and reference summaries for each MCU family.

The main book should contain the full learning narrative and deep explanations. This folder should remain more direct and family-oriented.

Suggested structure for each MCU family page:
- Overview
- Core architecture
- Typical strengths
- Typical weaknesses
- Peripheral profile
- Toolchain and ecosystem
- Common use cases
- Cost and availability perspective
- Comparison with other families
